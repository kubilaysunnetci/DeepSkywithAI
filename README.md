# DeepSkywithAI - MATLAB GUI Application

This repository contains the user-friendly MATLAB Graphical User Interface (GUI) application designed for the automated classification of astronomical objects (Galaxies, Stars, and Quasars) using SDSS photometric data. 

The application allows users to load unedited data in Excel format and yields high-precision astrophysical predictions in less than one second using top-performing deep learning models.
